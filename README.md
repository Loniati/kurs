# Student Management App

Учебный проект на C# (Avalonia UI), реализующий базовый функционал учета студентов.

## Стек технологий
* **C# / .NET**
* **Avalonia UI**
* **MVVM Architecture**

## Описание
Проект представляет собой приложение с использованием паттерна MVVM для управления списком студентов.

## Запуск
1. Склонируй репозиторий.
2. Открой `AvaloniaApplication1.sln` в Visual Studio.
3. Нажми `F5` для запуска.
